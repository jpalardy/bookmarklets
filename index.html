Jonathan's Bookmarklets

Original code found in my bookmarklets project on github.

To "install", drag a bookmarklet link into your browser toolbar.

To run, click on the bookmark while on the webpages it is indented to run on.

To "uninstall", delete the bookmark.

Bookmarklets

  • pagemeta — Grab page's title and url, dated today
  • toggle ruby — Toggle's the visibility of <rt> tags.
  • amazon-ca — Switch amazon.com links to amazon.ca.
  • crash! — Crash a Chrome tab: useful when things get out of hands.
